A Museum of Some Fractals...

Controls:
  - "W", "A", "S", and "D" keys = Movement up, left, down, right (respectively)
  - "+" (or "=" key) and "-" keys = zoom in and out (respectively)
  - up arrow, down arrow = more precision and less precision (respectively)
  - keys "1" through "4" = select different fractals

1: Mandelbrot Set 2: Broken Mandelbrot Set #1 3: Broken Mandelbrot Set #2 4: Burning Ship Fractal
